## 📥 Instalación en Local

1. **Clonar el repositorio**:

   ```bash
   git clone https://github.com/web-v2/rips-txt-json.git
   cd rips-txt-json
   ```

2. **Instalar dependencias y ejecutar**:

   ```bash
   npm install
   npm run dev
   ```
